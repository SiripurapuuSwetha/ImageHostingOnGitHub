## Overview

This repository contains images stored in the images/ folder. It is crucial to ensure that the repository remains up-to-date with correct and relevant images. Each image uploaded should be properly referenced in the associated Excel file to maintain consistency and accuracy

## How to Upload Images

To add new images to the repository, follow these steps:

    Step 1: Navigate to the images/ folder
    
    Step 2: Go to the images/ folder in this repository and Upload your image
    
    Step 3: Click on the Add file button and select Upload files.
    
    Step 4: Drag and drop or browse to select the image(s) you want to upload.
    
    Step 5: Click Commit changes to save the uploaded image(s) in the repository


## Get the Image Link and update the Excel file

After uploading the image, follow these steps to get the image URL:

    Step 1: Locate the uploaded image
    
    Step 2: Navigate to the images/ folder and find the newly uploaded image.
    
    Step 3: Click on the image file to open it.
    
    Step 4: Right-click on the image and select Copy image address OR click on the Download button and copy the URL from the browser address bar.
    
    Step 5:Open the Excel file that contains links to images in the repository.

    Step 6: Find the corresponding entry

    Step 7: Search for the row where the previous image was listed and Update the link

    Step 8: Replace the old image URL with the new one copied from the repository.

Save and commit the updated Excel file back to the repository.
